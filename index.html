Technische details

FoldersVergelijker is opgebouwd uit meerdere componenten:

📡 Scrapers

Node.js (puppeteer-extra + stealth) voor 9 stores. PHP cURL voor kaufDA.de (__NEXT_DATA__) en AlleFolders GraphQL API.

🗄️ Database

MariaDB 10.4.32. Tabellen: products, product_prices, stores, folders, categories. Upsert per dag.

🌐 Frontend

PHP public page met search, filters, sort, grid/list view. Shopping list met autocomplete en PDF export.

📧 Notificaties

Gmail SMTP via PHPMailer. PDF-billage via FPDF v1.86 met cp1252 encoding (iconv).
